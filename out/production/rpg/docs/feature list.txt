VIT (vitesse) -> + elle est elevee, - de délai entre les attaques
ATK (attaque) -> + elle est elevee + de degats
INT (intelligence) -> soit ATK pour les mages soit buff d'autres stats.
DEF (défense) -> + elle est élevée - de dégats lors d'une attaque.
MNA (mana) -> stat unique à mage ou sorcier
STM (endurance) -> + elle est élevé, + d'action possibles.
LCK (chance) -> la probabilité des évènements ayant une faible probabilité augmente.

etats speciaux:
0 = normal;
1 = paralysé -> endurance à 0 pendant 2 délais / passe deux tours sans pourvoir agir
2 = confus -> 50% de chances d'attaquer un allié ET 1/4 de chance de ne pas appliquer la commande demandée (donc atk random).
3 = aveuglé -> 75% de chance de miss l'attaque
4 = ralenti -> VIT diminuée de moitié
5 = poison -> 15 DMG a chaque action du joueur + dégats critiques obligatoires par les enemis.
